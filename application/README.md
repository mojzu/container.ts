# Application

TypeScript application template.

## Developer

```Shell
# Install dependencies.
$ yarn install

# Run tasks defined in `gulpfile.js`.
$ yarn run clean|distclean|test|lint|docs|start|inspect|build

# Use production flag for release builds.
$ yarn run build -- -p
```
