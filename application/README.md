# Application

TypeScript application template.

## Developer

```Shell
# Install dependencies.
$ yarn install

# Clean distribution directory.
$ yarn run clean

# Clean Node modules directory.
$ yarn run distclean

# Run tests with coverage.
$ yarn run test

# Run linter.
$ yarn run lint

# Build documentation.
$ yarn run docs

# Start application.
$ yarn run start

# Start application with Chrome V8 inspector.
$ yarn run inspect

# Build application binary.
$ yarn run build
```
